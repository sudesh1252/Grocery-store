STEP 1: Find Your MySQL Password
─────────────────────────────────

Your MySQL password is what you set when you installed MySQL.

Common default passwords:
• (empty - no password)
• root
• password
• admin
• mysql
• 123456


STEP 2: Test Your Password
───────────────────────────

Run this command and enter your password:

"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -p

If it works, you'll see:
mysql>

Type: EXIT


STEP 3: Update backend\.env
────────────────────────────

1. Open file: backend\.env

2. Find this line:
   DB_PASSWORD=

3. Add your password after the = sign:
   DB_PASSWORD=YourActualPassword

   Examples:
   DB_PASSWORD=root
   DB_PASSWORD=password123
   DB_PASSWORD=admin

   If no password (empty):
   DB_PASSWORD=

4. SAVE THE FILE!


STEP 4: Create Database
────────────────────────

Run this (replace YOUR_PASSWORD):

"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -pYOUR_PASSWORD -e "CREATE DATABASE IF NOT EXISTS shree_grocery;"

Example if password is "root":
"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -proot -e "CREATE DATABASE IF NOT EXISTS shree_grocery;"

Example if NO password (empty):
"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -e "CREATE DATABASE IF NOT EXISTS shree_grocery;"


STEP 5: Restart Backend
────────────────────────

1. Go to your backend terminal
2. Press: Ctrl + C (stops the server)
3. Run: npm run dev
4. Should see: "✅ MySQL Database Connected Successfully"
