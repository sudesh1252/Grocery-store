🚀 HOW TO RUN (SIMPLE):

STEP 1: Create MySQL Database
   mysql -u root -p
   CREATE DATABASE shree_grocery;
   EXIT;

STEP 2: Configure Backend
   Open: backend/.env
   Update: DB_PASSWORD=your_mysql_password

STEP 3: Run Backend
   cd backend
   npm install
   npm run dev

STEP 4: Run Frontend (New Terminal)
   cd frontend
   npm install
   npm start

STEP 5: Browser Opens Automatically
   URL: http://localhost:3000
   ✅ Start using the application!

   GET /api/invoices/stats - Dashboard stats
